# Графи пам'яті (FalkorDB)

Єдине джерело правди для скілів агента — c:\Antigarvity_workspace\.agent\skills\. Усі протоколи та схема посилаються на ці графи.

## Поточні графи

| Граф | Призначення | Запис (MCP) |
|------|-------------|-------------|
| **Grynya** | Основний (production). Усі сесії /db, /sa, /ss зберігаються тут. | За замовчуванням (GRAPH_NAME) |
| **Grynya_v2.0** | Рефакторинг схеми. Копія колишнього графа Cursa4; паралельна робота над новою схемою. | Тільки читання / експерименти (поки що) |
| Cursa4 (deprecated) | Знятий з експлуатації. Дані скопійовано в Grynya_v2.0. | Не використовувати |

## Міграція Cursa4 -> Grynya_v2.0

Граф Cursa4 було вручну розпочато; його повну копію перенесено в Grynya_v2.0 за допомогою інструменту MCP: mcp_falkordb_copy_graph(source_graph=\"Cursa4\", destination_graph=\"Grynya_v2.0\"). Після виконання копіювання пошук та експерименти з новою схемою проводяться в графі Grynya_v2.0. Щоденний запис сесій лишається в графі Grynya.

## Пошук (query_graph, research_graph)

Для пошуку по історії сесій використовуй граф Grynya. Для роботи над рефакторингом схеми можна додатково запитувати Grynya_v2.0.
